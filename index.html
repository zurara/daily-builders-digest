Apple Intelligence 向第三方开放——iOS 27 将支持用户自选 Claude、Gemini 等 AI 模型

据彭博社报道，Apple 正在内部测试 Google Gemini 和 Anthropic Claude 的集成，计划今秋在 iOS 27、iPadOS 27 和 macOS 27 中推出"Extensions"功能。用户将可以在设置中自由选择 AI 模型来驱动 Siri、写作工具、Image Playground 等所有 Apple Intelligence 功能。这标志着 Apple 从 AI 供应商角色转向 AI 分发平台。OpenAI 据报道对此不满，可能考虑法律行动。此举意味着 Anthropic 和 Google 将获得直达十几亿 iPhone 用户的通道。

📎 Bloomberg  ·  9to5Mac

02

OpenAI-Microsoft 合作重构——非独家许可、AGI 条款删除、多云开放

4 月底公布的新协议中，Microsoft 对 OpenAI IP 的许可变为非独家（延续至 2032 年），OpenAI 可在 AWS、Google Cloud、Oracle 等任何云平台提供服务。AGI 条款被删除，收入分成设上限。双方从"深度绑定"走向"战略合作但各自独立"，标志着 AI 产业格局从独家联盟转向多方竞争。

📎 CNBC  ·  OpenAI Blog

03

Cloudflare 裁员 1100 人——AI 使用增长 600%，营收创历史新高

Cloudflare 裁掉约 20% 员工（1100 人），CEO Matthew Prince 将原因直指 AI：内部 AI 使用量三个月增 600%，AI agent 已深入 HR、市场、财务等部门。同周 Upwork 裁 24%（145 人），Coinbase 等也在裁。特殊之处在于：企业在盈利状态下裁员。斯坦福 AI Index 报告称生成式 AI 三年达 53% 渗透率，超过 PC 和互联网。

📎 TechCrunch  ·  Fast Company

04

Garry Tan 开源 GBrain——给 AI Agent 装上长期记忆的八层知识系统

Y Combinator 掌门人 Garry Tan 开源了 GBrain，一个给 AI agent 提供长期记忆的知识图谱系统。八层架构将 Markdown 笔记变为自动连线的知识网络，底层用 Git + PostgreSQL/pgvector 做混合搜索。最酷的"dream cycles"功能让 agent 在用户睡觉时自动整理知识库。MIT 许可，发布首日 5400 星，目前超 14000。社区争议：核心功能是 Markdown 指令文档而非可执行代码——但这本身或许正是 AI 编程的新范式。

📎 Garry Tan on X  ·  GitHub
