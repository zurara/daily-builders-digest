Karpathy 宣告：HTML 才是 AI 的默认输出格式，Markdown 时代该结束了

Andrej Karpathy 在 X 上发帖——一万三千赞——提出 AI 输出格式的进化路线：纯文本 → Markdown → HTML → 交互式神经视频。人类大脑有三分之一是视觉处理器，是信息进入大脑的「十车道高速公路」。HTML 能做表格布局、SVG 图表、JavaScript 交互，而 Markdown 只是略好的纯文本。Anthropic 工程师 Thariq Shihipar 同期发文「HTML 的不合理有效性」，宣布完全弃用 Markdown。Salesforce Agentforce 已用「自适应响应格式」将 LLM 输出自动转成轮播卡片和富按钮——未来产品的竞争力在于答案以什么形态到达用户。

📎 Andrej Karpathy — X

02

Cursor SDK 公测：编程 Agent 变成可编程基础设施

Cursor 将 SDK 推进公测——一个 TypeScript API，让开发者用几行代码创建、运行和管理编程 Agent。支持本地或云端 VM 运行，按 token 消耗计费。早期用户已在用它自动修 bug、做代码摘要、故障根因分析。开发者工具的终局不是更好的 IDE，而是 Agent 基础设施——就像 Stripe 不是支付页面，而是支付基础设施。

📎 Build programmatic agents — Cursor Blog  ·  DevOps.com

03

Figma Make Skills：设计系统和 AI Agent 直接握手

Figma 五月更新发布 Make Skills——用 Markdown 文件定义可复用的设计规范，斜杠命令调用即自动遵守设计系统。MCP 服务器支持 Claude Code、Cursor 等十多个客户端，/prototype-to-figma 可将本地原型搬到 Figma 画布并关联设计系统组件。Make 还登陆 iOS/Android，真实手势预览。原型验证从「天」降到「分钟」。

📎 Figma Release Notes — May 2026  ·  UX Collective

04

Long Lake 63 亿美元收购 Amex GBT：世界首笔 AI Take-Private

Long Lake Management 以 63 亿美元收购美国运通全球商务旅行——世界首笔 AI 驱动的 take-private 交易。其横向 AI 平台 Nexus 80% 基础设施跨行业共享，针对每家公司定制部署。创始人 Taubman 说，过去部署要一年，现在几天见效。收购后不裁员而驱动增长——HOA 业务年增长从 0-5% 跃升到 20%+。AI 产品的价值不在于好用的工具，而在于更好的业务结果。

📎 Long Lake Acquires Amex GBT — BusinessWire  ·  No Priors Podcast
