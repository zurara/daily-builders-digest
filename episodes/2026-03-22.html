🎙 播客对话 · 大雨 × 小苇
大雨
Karpathy 在这集里用了一个词——"loopy era"，循环时代。意思是，AI 正在从"你给一个问题、它给一个答案"的模式，进入"不断循环、自我改进"的模式。
小苇
具体来说是什么意思？
大雨
AutoResearch 是他放出来的开源项目。给 agent 一个简单的 LLM 训练代码库，让它自己去改、自己跑5分钟实验、看结果有没有变好、保留或者丢弃这次改动，然后继续。这个循环跑两天——700次实验，发现了20个有效优化，包括一些人类没想到的架构改进，比如 QK Norm 和 RoPE 的重新排序。
小苇
听起来就像你雇了一个永远不睡觉、永远在改代码的工程师。
大雨
对，而且这个"工程师"失败了不会沮丧，成功了不会骄傲，就是一直在跑。Karpathy 用的比喻是"SETI@home for AI"——以前大家把闲置算力贡献给搜索外星人，现在可以贡献给 AI 自我训练。他觉得这种模式将来会成为前沿实验室的标配。
小苇
Fortune 最近也写了一篇文章叫《The Karpathy Loop》，反响很大。
大雨
对。两天700次实验这个数字出来，很多人都在讨论：我们是不是已经进入了"AI 研究员比人类研究员更高效"的阶段？

Karpathy 自己的说法是：coding agents 现在是"近在眼前的杠杆点"。他不是在说五年后——是现在。代码 agent 已经能处理真实任务，懂得怎么用好它的人，会有巨大的生产力优势。
小苇
他也聊到了就业市场的变化……
大雨
这部分他说得比较克制。他的判断是：会有巨大的结构性变化，但同时会出现很多新类型的工作——比如"给 agent 设计实验框架"本身就是一个新工种。

还有他聊到的"model speciation"——模型物种分化。他认为不同的模型会进化成专门适配不同任务的形态，不是所有任务都用同一个大模型。就像生物进化一样，通才和专才会同时存在，各有生态位。
小苇
还有一段关于"MicroGPT"和 agentic education，感觉是最有未来感的部分。
大雨
他在描述的是：不只是工程师用 agent 来写代码，而是整个学习和研究的方式都会被 agent 重塑——你不再只是学知识，你在学怎么"编排 agent 来学习"。这是一个很大的认知跳跃。
小苇
这集对 builder 来说非常值得认真听，特别是如果你在做 AI 工具、或者在思考怎么在 agent 时代找到自己的位置。
核心观点 · Karpathy Loop
THE LOOPY ERA
01
AI 的下一阶段不是"更好的回答"，而是"自主实验的循环"——agent 写代码、跑训练、评估结果、自我迭代，无需人类介入。
02
AutoResearch 在单卡上两天跑了700次实验，发现了20项优化，其中包括人类工程师没有想到的架构改进。
03
Coding agents 是"近在眼前的杠杆点"——不是五年后，而是现在。懂得编排 agent 的人将拥有不对等的生产力优势。
04
模型物种分化（model speciation）：未来不是一个大模型统治一切，而是不同模型进化出各自的生态位——通才与专才并存。
