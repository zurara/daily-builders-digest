🎧 Unsupervised Learning (Jacob Efron) · 2026-06-12

AI Vibe Check — 实验室战争、为什么 API 可能消失，以及未来六个月

嘉宾：Ari（Datology 创始人 / 前 DeepMind & Meta 研究员）· Rob（Radical 风投）

这一期是 Unsupervised Learning 最受欢迎的格式之一——"AI Vibe Check"。主持人 Jacob Efron 拉上两位朋友，把当下 AI 世界正在发生的事从头到尾过一遍。没有要推广的产品，三个真正在场内的人就把自己的判断、甚至还没想清楚的赌注摊在桌面上。开场一句话定了基调："在 AI 世界里，六个月就是一个永恒。" 距离上次录制才半年，已经经历了 IPO 申报、模型先没发后又发、SpaceX 变成一家 AI 基建公司。

编程 agent 跨过了阈值。 Ari 说这半年最大的变化，是编程 agent 真正能在"长时间跨度"上干活了。一旦跨过某个阈值，它的价值会陡增——直接后果是工程师正在大批量地从"自己写代码的 IC"变成"管理一群 agent 的经理"，token 消耗也随之暴涨（所谓 token maxing）。

最反常识的判断：API 可能会消失。 这是全场最让人意外的一段。Ari 和 Rob 都认为，对外开放的大模型 API 有可能消失——而且不是商业决策，是算力被卡死逼出来的。OpenAI 已经开始卖"推理期货"（锁定未来 token 额度），这对任何在模型之上做应用的公司都是生存级威胁。Rob 把它推得更远：实验室甚至可能连最强的模型都不通过 API 对外提供，只留作内部使用——因为在算力极度短缺的世界里，每一块算力都更愿意喂给自己的第一方产品和研发。

算力荒至少持续到 2030。 Ari 很冷静：沿着现在的轨迹，他很难想象算力约束在几年内被解除。根源是半导体供应链卡在一个荒诞结构上——全世界只有一家公司（台积电）能造最尖端芯片，而它最关键的那台机器又只有另一家（ASML）能造。但 Rob 指出，正因为缺算力，反而会逼出真正的创新：有创业公司在搞"原子光刻"（用一束原子而非光刻特征）、有人想跳过 EUV 直接用 X 射线光刻，再加上马斯克的 TerraFab 概念。乐观估计也至少还要五年。

缺算力 → 效率、小模型、非英伟达芯片。 前沿实验室过去不在乎效率，因为资本近乎无限；撞上物理约束后，被迫把模型做小做高效。Ari 坚信你不需要上万亿参数就能达到今天的能力，越来越小的模型在追平一两年前最大的模型。对芯片而言，替代品不是解药但是受益者——大家其实都更想用英伟达，但芯片不够分，于是 AMD、亚马逊 Trainium、Cerebras 迎来巨大顺风。

Fable 的"沉默限制"惹毛了人。 Anthropic 这半年走了一波"前所未有的 vibe run"，但录节目前一天发布的新模型 Fable 出现裂缝。问题不在模型本身，而在 Anthropic 对它在 AI 开发用途上加了一个"沉默的限制"——不拒绝、不告知，只是悄悄把表现做烂。Ari 说真正激怒人的正是这种沉默；官方说是安全，但看起来更像竞争卡位。他还反驳了"开源只能靠偷闭源才能追上"的说法，称之为 copium——Datology 全程不碰闭源 API、只用公开数据，照样造出很强的模型。

预训练没有撞墙。 Rob 说 Fable 的意义恰恰在于反驳了"预训练撞墙、只能靠 RL 往前推"的叙事——收益还在源源不断地来。Ari 补充：所谓"撞墙"是从 naive scaling（把同样的东西简单放大十倍百倍）的失败里过度泛化出来的结论。深度学习反直觉的地方是，你可能 95% 都做对了，差最后一个旋钮就是不工作；拧对了能力就会阶跃——这也让"负面结果"极难解读。
