01 · 头条

Anthropic 发布三大 Agent 新功能——Dreaming 让 AI 从经验中学习，Outcomes 配质检员，多智能体编排处理复杂任务

5月6日的 Code with Claude 大会上，Anthropic 宣布了 Claude Managed Agents 的三项重磅升级。Dreaming（做梦）让 Agent 在任务间隙回顾历史记录、提取模式，实现跨会话学习——就像大脑在睡眠中整理记忆。Outcomes（成果验证）引入独立评分 Agent，在隔离的上下文窗口中按用户定义的标准检查输出质量，测试显示任务成功率提升约 10 个百分点。多智能体编排则允许主 Agent 将复杂任务拆分给多个专业子 Agent，每个拥有独立模型、提示词和工具集。三项功能组合起来，Anthropic 正在构建一个完整的 Agent 操作系统。

📎 Anthropic Blog  ·  SiliconANGLE  ·  Digital Trends

02

Boris Cherny：编程正在经历"印刷术时刻"——手机写代码，一天 150 PR

Claude Code 创造者 Boris Cherny 在 Code with Claude 大会上分享了他的极致工作流：主要在手机上编程，白天运行数百个 Agent，夜间数千个。他每天提交几十个 PR，上周创下一天 150 PR 的记录。更深远的是他的历史类比——十五世纪印刷术让识字率从 10% 升到 70%，编程正经历同样的民主化。他的团队里每个人都写代码，包括设计师、产品经理和财务。

📎 Training Data Podcast

03

DeepSeek V4 Flash：284B 参数仅激活 13B，MIT 开源，价格不到主流模型零头

DeepSeek 发布 V4 Flash 混合专家模型——总参数 284B 但每次推理仅激活 13B，支持百万 token 上下文。输入 $0.14/M tokens、输出 $0.28/M tokens，MIT 许可证开源。推理和编码表现不错，但在最复杂的 Agent 工作流上仍有差距。对注重成本的开发者来说，这是一个极具竞争力的选择。

📎 Hugging Face  ·  Simon Willison

04

Dario Amodei：Anthropic Q1 增长 80 倍——"我们要买下所有能买到的算力"

Anthropic CEO Dario Amodei 在 Code with Claude 大会上透露，2026 年 Q1 使用量和收入实现 80 倍增长，主要由 Claude Code 和 Agent 工作流驱动。这种增长带来了真实的基础设施压力——Claude Pro/Max 用户此前体验到的稳定性问题正源于此。Dario 表示团队正全力获取更多算力。他还分享了产品哲学：为指数级增长做规划，因为当前模型做不到的事下一代可能就行。
