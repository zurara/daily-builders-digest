Story 01

NVIDIA RTX Spark：从"工具"到"队友"，PC 正在被重新定义

Computex 2026 上最大的产品发布。NVIDIA CEO 黄仁勋发布了 RTX Spark 超级芯片——把 Blackwell GPU 和全新 ARM 架构 Grace CPU 融合在一起，最高 128GB 统一内存，1 petaflop AI 算力。你可以在笔记本上本地跑 1200 亿参数的大模型，100 万 token 上下文窗口，不需要云。NVIDIA 的定位是四个字：Tool to Teammate——"工具变队友"。他们跟微软深度合作，要把 Windows 变成 AI Agent 操作系统。RTX Spark 不只让你用 AI，而是让 AI Agent 在你的电脑上自主运行。今年秋天上市，ASUS、Dell、HP、联想、微软 Surface、MSI 都会出搭载 RTX Spark 的机型，预计超过 30 款。PC 作为产品的定义变了——从"你完成任务的设备"到"能自主工作的团队成员"。

Story 02

Dell XPS 13 起价 $599：Premium 硬件的价格下沉

同样在 Computex 上，Dell 发布了全新 XPS 13——学生版 599 美元，普通版 699 美元。这是 Dell 史上最轻最薄的 XPS：0.9 公斤，12.7 毫米厚，CNC 铝合金机身，2.5K 触摸屏，四扬声器杜比全景声，1080P 摄像头支持 Windows Hello。完全正面对标苹果 MacBook Neo，但给了触摸屏和更好的音频系统——这些是 MacBook Neo 没有的。一个经典的"价格下沉"产品策略：用 Premium 品质打低价市场，在 AI PC 时代抢占入门用户。

📎 Tom's Hardware · Windows Central

Story 03

Merge Agent Handler：谁来管住你公司的 AI Agent？

公司里每个人都在用 Claude、ChatGPT、各种 AI Agent 连接公司系统，但 IT 部门不知道谁在用什么、能访问哪些数据。Merge 发布 Agent Handler for Employees 解决这个问题：连接身份认证系统，导入员工和组，设定每个人可以用哪些 AI 工具、连接哪些系统、执行哪些操作。所有 AI 会话都有日志，加上数据防泄漏控制。六月二号 Agent Handler 宣布上架微软 Agent Store，支持 MCP 协议。AI Agent 正在从"个人效率工具"变成"企业基础设施"——Merge 看到了"管理 Agent 的中间层"这个机会。

📎 SiliconANGLE · Merge

Story 04

PM 的 K 型分化与 AI 时代的竞争壁垒

产品经理角色正在经历 K 型分化——一部分人往上走（AI Product Manager 和 AI-Powered PM），一部分人往下走（传统通才型 PM，其核心工作正被 Agent 吞噬）。团队配比也在变：从 1 PM : 4 工程师，到前沿团队的 2 PM : 1 工程师——Agent 写代码，但需要更多人做产品策略和用户研究。关键问题：当竞争对手也有同样的 AI，你的竞争壁垒在哪？答案可能是——壁垒不再是技术本身，而是你对用户的理解深度、数据的独特性、以及团队使用 AI 的方法论。
