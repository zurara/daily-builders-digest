OpenAI Codex 桌面大更新：Background Computer Use 让 AI 直接操控你的 Mac

4 月 16 日，OpenAI 发布 Codex 桌面应用的重大更新。全新 Background Computer Use 功能让 Codex 可以看到屏幕、移动光标、点击和输入——直接操控 macOS 上的任何应用。多个 Agent 可在后台并行工作，不干扰用户前台操作。 配套发布 90+ 新插件（Atlassian、CircleCI、GitLab、Microsoft Suite 等）、记忆功能（从过去操作学习偏好）、内置浏览器和 SSH 远程开发支持。AI 助手的战场从对话框扩展到整个桌面——桌面应用的产品形态即将被重新定义。

📎 Codex for (almost) everything — OpenAI  ·  Codex Desktop Major Update — SmartScope

02

Claude Cowork GA + Managed Agents：Anthropic 的企业 Agent 基础设施

4 月 8 日，Anthropic 三连发：Opus 4.7 模型、Cowork 面向所有付费用户开放、以及 Managed Agents 公测。Managed Agents 提供沙盒执行、状态存档、权限管理和全链路追踪——从原型到生产"天而非月"。定价 $0.08/session-hour + token 费。Notion、Asana、Sentry 为首批采用者。OpenAI 从桌面入手，Anthropic 从云端入手——都在抢 Agent 时代的操作系统位。

📎 Anthropic launches Managed Agents — SiliconANGLE

03

Adobe × NVIDIA：CX Enterprise Coworker，营销人的 AI 同事

Adobe Summit 2026 发布 CX Enterprise 及与 NVIDIA 合作的 CX Enterprise Coworker——不叫"助手"叫"同事"，自主处理内容创作、客户体验编排、跨渠道投放。NVIDIA 提供 Nemotron 模型和安全运行时，Adobe 提供 CX 治理层。Jensen Huang 与 Shantanu Narayen 同台 Keynote。Agent 的命名策略正变得重要：Coworker 暗示更高自主性，垂直 Agent 时代到来。

📎 Adobe Unveils CX Enterprise Coworker — Adobe Newsroom

04

Aaron Levie：从聊天到执行，企业需要 Agent Deployer

Box CEO Aaron Levie 在 Fast Company 采访中指出：企业 AI 正从"聊天时代"走向"Agent 执行时代"。Box 四月初发布 Box Agent 和更新的 AI Studio，同时 Levie 定义了全新企业角色"Agent Deployer"——专门识别团队中 ROI 最高的工作流，用 Agent 加速 100 倍。不是开发者，不是 PM，而是一个全新运营职能。当 Agent 能处理线索、审合同、客户上手，产品经理的价值在于判断和决策。
