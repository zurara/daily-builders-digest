📰 OpenAI · 2026-06-11

OpenAI 收购 Ona — Codex 从"帮你写代码"进化成"自己有工位的持久 agent"

每周 500 万人在用 Codex · 较今年早些时候 +400%

6 月 11 日，OpenAI 宣布要收购 Ona——一家做安全、持久运行环境的云基础设施公司。表面是一笔收购，本质是 Codex 的一次变形：OpenAI 想让 Codex 不再只是"你打开浏览器、它帮你写两行代码"，而是能在你关掉电脑之后，继续在云端干上几个小时、甚至几天的活。

关键在"工位可以开在企业自己的云里"。 Ona 的技术让 agent 在客户自有的云环境中运行，公司能继续掌握数据访问、安全策略、凭证、日志、合规这些东西，同时又用上 Codex。很多大企业过去不敢用，卡的就是这一关——这次正是冲着企业级落地去的。

这就是本周的主线：模型在吃掉脚手架。 持久运行、云上长时任务，这些原本是你要在模型外面自己搭的框架，现在被直接收进了产品本体。Sam Altman 当天只发了一句 "really looking forward to working together"，配的就是 Ona 团队，两千多赞——建造者圈把这当成 Codex 进入"持久 agent"时代的明确信号。

Meta Superintelligence Labs · 本周

Meta 发布 Muse Spark — Muse 家族首个模型，能并行调度多个推理 agent

原生多模态推理 · 视觉思维链 · 多 agent 编排

Muse Spark 主打原生多模态推理、工具使用，还有一个叫 Contemplating（沉思）的模式——会同时调度好几个推理 agent 并行去想同一个问题，等于把一个团队塞进一个模型里。成绩上在 Humanity's Last Exam 拿到 58%、FrontierScience Research 38%。又是"模型把外部多 agent 框架内化"的同一条主线。benchmark 归 benchmark，真好不好用还得等大家上手。

NVIDIA · 本周

英伟达发布 Nemotron 3 Nano Omni — 开源全模态推理模型，30B MoE

视觉 + 音频 + 语言统一 · 吞吐量最高 9 倍

不是 Cosmos，而是一个开源的全模态推理模型，把视觉、音频、语言塞进一个 300 亿参数的 MoE 架构里，号称吞吐量比同类开源多模态模型高出最多 9 倍。它跟前两周的 Cosmos 3 是一条线——英伟达在认真推开源的全模态底座，想让更多人能在便宜硬件上跑得动统一处理图像、声音、文字的模型。对想自己折腾的人是好消息。
