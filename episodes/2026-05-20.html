Story 01

GitHub Copilot App：代码 Agent 有了自己的家

5月14号，GitHub 正式发布 Copilot App 技术预览版——不是 VS Code 插件升级，而是一个全新的独立桌面应用，支持 Mac、Windows 和 Linux。核心理念是「从 Issue 出发的全流程 Agent」：打开一个 GitHub Issue，Copilot 自动分析需求，创建独立的 git worktree 和分支，在隔离环境里写代码，最终输出 Pull Request。你可以同时跑多个 Agent session，并行处理不同任务。目前对 Copilot Pro/Pro+ 用户开放早期试用。开发工具竞争已经从「编辑器里的补全」升级到了「独立的代码 Agent 平台」。

📎 GitHub Blog · The New Stack

Story 02

Googlebook：Chromebook 终结，Android 笔记本时代开始

Google 在 The Android Show 上宣布 Googlebook——全新品类的 Android 笔记本，围绕 Gemini AI 从底层设计。告别 Chromebook 的功能局限，直接跑所有 Android 应用。独特设计包括 Magic Pointer 智能推荐和 Glowbar 发光条。Acer、ASUS、Dell、HP、Lenovo 均加入，预计今秋上市。关键信号：操作系统不再是壁垒，AI 集成才是差异化。

📎 Google Blog · TechCrunch

Story 03

Amazon Alexa Podcasts：AI 生成播客，按需制作

亚马逊在 Alexa+ 上线 Alexa Podcasts——说一个话题，AI 搜集信息、生成大纲，你可以调整长度和语气，然后用双 AI 主播录制。亚马逊与 AP、Reuters、华盛顿邮报等200多家媒体签约，提供真实新闻源支撑。从 NotebookLM 到 Alexa Podcasts，AI 播客正在把「按需信息获取」变成音频形式。内容正从「人们创作的东西」变成「AI 按需生成的服务」。

📎 TechCrunch · Variety

Firefox Shake to Summarize 登陆 Android

Mozilla 的 Shake to Summarize 正式登陆 Android——摇一下手机或点菜单即可获得5000字以内网页的 AI 摘要。底层使用 Mozilla 云端 Mistral-Small 模型，而非依赖 Big Tech。去年拿下 TIME Best Inventions 特别提名。产品启示：怎么在不牺牲品牌价值观的前提下，把 AI 做进产品。
