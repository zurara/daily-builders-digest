Figma Weave + Make Mobile：产品设计工具全面迎接 AI 原生工作流

Figma 本周发布重磅更新。全新 Figma Weave 让设计师在可视化画布上构建可重复、可扩展的生成式 AI 工作流——输入产品描述即可自动生成符合品牌规范的营销图、社交 post、landing page 原型。OpenAI ChatGPT Images 2.0 模型已直接集成，支持 Design、Draw、Slides、Buzz 和 FigJam。 同时矢量编辑速度提升 10 倍，Make 帧率提升 4 倍，Make 登陆 iOS 和 Android。企业级新功能 Developer Logs 和 Plan Access Tokens (PLANTs) 正在为"Agent 访问 Figma"铺路——产品设计的自动化即将再上一个台阶。

02

Linear Agent + MCP：项目管理工具变成 Agent 的大脑

Linear Agent 现在支持 MCP 连接，可以接入 Granola、Glean、Notion、PostHog 等外部工具——获取会议记录、企业知识库、用户行为数据，基于完整上下文做调查、写 spec、更新项目。Issue 页面还能直接启动 Claude Code、Codex、Copilot 等编码工具。产品管理工具的竞争维度变了：不再比 UI，而是比谁能成为 Agent 的最佳入口。

03

Agent-Led Growth：从 SEO 到 ALG，营销的最大平台转移

Training Data 播客采访 ProFound CEO James Cadwallader：互联网没变，变的是谁在用——Agent 替用户做购买决策。人类搜索时前五个链接拿走 95% 注意力，Agent 会用 65 个网页回答一个问题。不同 AI 平台推荐品牌截然不同：Gemini 偏爱 YouTube，ChatGPT 偏爱 Reddit，Claude 更多使用实时网络。产品需要对 Agent 友好——清晰的 API、完整的文档、可解析的信息架构。

04

Solo AI 建造者月入百万的产品策略

Peter Yang 分享 solo AI founder Tibo Maker 的经验——月营收超 $1M。关键策略：第一天就收费验证需求；跟着用户意外行为转型（Typeframe → Revid，现月收入 $600K+）；定价 $50–100/月过滤白嫖；月流失率控制在 20% 以下；做 100+ tool page 覆盖长尾搜索意图。核心：不要做万能 AI 工具，要做解决具体痛点、用户愿意付费的工具。
