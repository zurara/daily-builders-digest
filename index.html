Google 搜索框 25 年最大改版——AI Agent 全面嵌入，10 亿月活用户

Google 在 I/O 2026 上发布了搜索框自诞生以来最大的一次重新设计。新搜索框支持完整句子输入，支持图片、视频和 Chrome 标签页等多模态输入，背后运行的是 Gemini 3.5 Flash。Google 同时推出「信息代理」（Information Agents），可在后台全天候监控互联网更新并主动推送。Generative UI 功能可根据查询实时生成交互式图表和模拟器，今年夏天对所有人免费开放。AI Mode 月活已超 10 亿。

📎 Google Blog  ·  The Next Web

02

Microsoft Copilot Studio「计算机使用」代理正式 GA——AI 像人一样操作界面

微软宣布 Copilot Studio 的计算机使用代理正式上线。代理通过视觉推理导航真实 UI 界面，不依赖选择器或脚本，布局变化时自适应。Graebel 公司已用它操作 Global Connect 系统完成交易，无需 API 集成。同时推出 Agent 365 治理平面，构建超过 5 个代理的企业需使用，每用户每月 $15。

📎 Microsoft Tech Community  ·  DevOps.com

03

Anthropic × 盖茨基金会 2 亿美元合作——AI 深入医疗、教育、农业

Anthropic 与盖茨基金会于 5 月 14 日宣布 2 亿美元、为期四年的合作，将 Claude 部署到全球医疗（脊髓灰质炎、HPV、子痫前期）、教育（K-12 循证辅导、非洲和印度基础读写）和农业（小农经营生产力优化）领域。全球约 46 亿人缺乏基本医疗服务。

📎 Anthropic  ·  Gates Foundation

04

YC CEO 几小时微调 Qwen3.5-397B——开源 AI 民主化加速

Y Combinator CEO Garry Tan 使用 Thinking Machines 平台在几小时内完成了 Qwen3.5-397B（近 4000 亿参数多模态模型）的微调。推文近 1000 赞。阿里出品的 Qwen 系列在全球开源社区中影响力持续扩大，配合 OpenAI 与微软伙伴关系重组（非排他、多云分发），AI 生态从平台锁定走向开放竞争。

📎 Garry Tan on X
