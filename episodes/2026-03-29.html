🎙 播客对话 · 大雨 × 小苇
大雨
Latent Space 这集真的值得好好讲。David Singleton 在 Stripe 做了很多年 CTO，帮公司在全球建了一套高度可靠的支付基础设施。很多人以为他下一站是另一家金融科技巨头，结果他选择了一个听起来很奇怪的方向：做一个消费者级别的 AI agent 操作系统。
小苇
他为什么选这个？
大雨
他在节目里说，他看到了一个结构性的机会：工具已经具备了，但中间缺一层。大多数人没有能力用好 LLM 来编排复杂的 agent 任务，但他们是有需求的。Dreamer 想做的就是那一层——用自然语言来发现、构建、使用 AI agent。
小苇
他们有一个叫"Sidekick"的东西，听起来很有意思。
大雨
对，Sidekick 是核心。它的定位是"agent 里的 agent"——一个帮你定制其他 agent 的 AI。你告诉它你想要什么，它去帮你组装工作流，而不是你自己去写 prompt 或者配置工具。
小苇
这个复杂度很高吧？让 agent 来创造 agent……
大雨
非常高。David 在节目里非常坦诚地讲了这里面的难点——context 管理、tool 调用的可靠性、用户信任的建立。他有一句话我觉得很重要："Reliability is not a feature, it's the product." 在 Stripe，可靠性是护城河，在 Dreamer，他把同样的价值观带过来了。
小苇
然后录音结束 11 天后，Meta 就宣布把整个 Dreamer 团队签走了，加入 Meta Superintelligence Labs？
大雨
是的。而且不只是人才收购，David 本人澄清说，他们的技术也一并授权给 Meta 使用。这不是那种"我们来你公司但原来的东西就没了"的故事，而是技术和团队一起转移。
小苇
所以这集播客就成了 Dreamer 以独立公司身份发声的最后记录。
大雨
正因如此，它有一种特别的重量感。你在听的时候知道：这个团队录音时还在想怎么做 growth，十一天后就要完全换一个轨道。那种"事情正在发生"的感觉，透过录音都能感受到。
小苇
对 builder 来说，这集有什么值得带走的？
大雨
两点。第一，agent 的核心不是"能做多少事"，而是"用户有多信任它"——这是 David 在 Stripe 积累的最深的直觉，也是他带进 Dreamer 的。第二，这次 Meta 的动作也在提醒我们：agent OS 这个赛道，大公司已经开始认真对待了。如果你在做这个方向，窗口期在收窄。
核心观点 · Dreamer × David Singleton
THE AGENT OS MOMENT
01
工具层已经成熟，缺的是中间那一层：让普通人能用自然语言编排复杂 agent 任务的操作系统。Dreamer 就是这个赌注。
02
Sidekick = "agent that builds agents"：用户告诉它想要什么，它负责组装工作流。复杂度极高，但这是真正的差异化。
03
"Reliability is not a feature, it's the product." David 把 Stripe 的核心价值观直接带进了 Dreamer——这是一个来自金融科技的世界观。
04
录音结束11天后，Meta Superintelligence Labs 收购整队。Agent OS 赛道的窗口期正在收窄——大公司已经下注。
