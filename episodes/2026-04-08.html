Figma AI Agent 进驻画布：MCP Server + Skills 系统重新定义设计工作流

Figma 四月更新正式推出 MCP Server 支持——AI Agent 不再是「在外面画好贴进来」，而是真正进入你的设计系统：读取 design tokens、组件库、spacing 规则，直接在 Figma 文件里创建和修改元素。 核心机制是 Skills 系统：团队用 Markdown 文件定义 Agent 的工作规范——组件复用逻辑、间距约定、风格变量。Agent 还具备「自愈」能力：截图检查产出，发现偏差自动修正。 更关键的是双向工作流已打通——代码渲染的 UI 可推到画布变为可编辑 frame，设计上下文也能拉回代码。Cursor、Warp、Factory、Firebender、Augment 均已对接。加上 Make Kits 让设计系统团队可以将代码指南打包进 AI 原型生成流程，这是真正的 design-to-code 闭环。

02

Softr AI Co-Builder：一句话，一套 Production-Ready 系统

Softr 从 no-code 工具平台转型为 AI-native 业务软件生成平台。AI Co-Builder 让非技术用户用自然语言描述需求——数据库、界面、权限、业务逻辑一次性生成，直接上线。核心卖点是「demo 级到生产级」的跨越：安全性、真实数据接入、权限管理一步到位。已服务超 100 万开发者、7000+ 组织，包括 Netflix、Google、Stripe。PM 的工具箱正在急速扩大。

📎 Softr launches AI-native platform — VentureBeat

03

Rocket「Vibe 麦肯锡」：$250 一份 AI 策略报告

印度创业公司 Rocket 发布 1.0 平台，从 1000+ 数据源（Meta 广告库、Similarweb API、自建爬虫）生成产品策略文档：定价、单位经济模型、GTM 建议全套。$25/月起做应用，$250/月出「麦肯锡级」报告。传统咨询要几十万美元，Rocket 把策略研究的成本压到三位数。Accel、Salesforce Ventures 已投 $15M 种子轮。大多 AI 工具帮你「做」，Rocket 帮你「想」。

📎 AI startup Rocket offers vibe McKinsey-style reports — TechCrunch

Codex 周活突破 300 万：构建者的引力场正在形成

Sam Altman 在 X 宣布 OpenAI Codex 周活跃用户突破 300 万，重置所有用户使用额度，并承诺每增长 100 万用户再重置一次，直到 1000 万。300 万人每周在用 AI 写代码做产品——工具、方法论正在趋同，AI 开发生态的网络效应正在加速。对产品团队来说，这不是「要不要用」的问题，而是竞争对手已经在用了。

Extreme Harness Engineering for Token Billionaires: 1M LOC, 1B toks/day, 0% human code, 0% human review

OpenAI 前沿团队的 Ryan Lopopolo 分享了「harness engineering」的概念——当 AI 模型足够强大时，关键不再是写代码本身，而是构建正确的「线束」让模型高效工作。他的项目 Symphony 管理着 100 万行代码、日均消耗 10 亿 tokens，零人工代码、零人工审查。核心洞察：要用系统思维不断追问「Agent 在哪犯错？我的时间花在哪里？如何把这部分自动化掉？」——然后一步步建立对自动化的信心。
