Cursor 3：Agent-First IDE，开发者从写代码变成了指挥 Agent

Cursor 发布了从头重建的第三代产品，核心逻辑发生了根本转变：IDE 不再假设开发者写代码，而是假设大多数代码由 AI Agent 完成，开发者的工作是编排和指挥。 最大变化是 Agents Window——全屏工作区，可同时管理多个 AI Agent，支持跨 repo、跨环境（本地 / worktree / 云端 / 远程 SSH）并行运行。底层由自研 Composer 2 模型驱动，专用编码数据训练，更快更省 token。 Cursor CEO 表示：模型、产品、运行时三块基础已到位，下一步是更自主的 Agent 和更好的团队协作。这不是功能更新，这是对「开发者工具应该是什么样」的重新定义。

📎 Meet the new Cursor — Cursor Blog  ·  Cursor 3 Agent-First Interface — Medium

02

Vercel 开源 Open Agents：云端编码 Agent 平台，人人可用

Vercel CEO Guillermo Rauch 宣布开源 Open Agents——完整的云端编码 Agent 参考平台，包含 Web UI、Agent 运行时、沙盒编排和 GitHub 集成。过去三个月 Vercel 周部署量翻倍，超 30% 由编码 Agent 发起（半年前不到 3%）。配套推出 Skills.sh 标准化 Agent 命令和 40+ React 最佳实践规则。经典的「开放标准，锁定基础设施」产品策略。

📎 Vercel Open Agents — GitHub

03

OpenAI 收购 Hiro Finance：从通用平台走向垂直产品

OpenAI 本周宣布收购 AI 个人理财创业公司 Hiro Finance，定位「AI 个人 CFO」，背后投资方包括 Ribbit Capital、General Catalyst。继去年收购 Roi 之后，这是 OpenAI 在个人金融赛道的第二次出手。Hiro 团队加入 OpenAI，产品 4/20 关停。信号明确：平台方正在从通用能力走向垂直产品深度——做 AI 理财的竞争对手，可能不只是其他创业公司了。

📎 OpenAI acquires Hiro — TechCrunch

04

SaaS 行业变天：从卖座位到卖结果，Agent Deployer 新工种诞生

Deloitte 和 Fortune 本周深度分析 SaaS 行业转型：AI Agent 正在摧毁传统按座位收费模型。AI-native 应用企业支出暴涨 108%，大企业更达 393%。Deloitte 预测到年底 40% 企业应用将嵌入任务级 AI Agent（当前不到 5%）。Box CEO Aaron Levie 指出企业里正在出现全新角色「Agent Deployer」——专门负责在组织内部部署和管理 AI Agent。垂直 SaaS 优势放大，数据本身正在成为可变现的产品。
